# REQ-F-download-audiobook: Download Audiobook Files

**Type**: Functional

**Status**: Draft

**Priority**: Should-have

**Source**: [US-download-audiobook](../user-stories/US-download-audiobook.md)

**Source stakeholder**: [STK-end-user](../stakeholders.md)

## Description

The system shall allow the user to download an audiobook's audio files in MP3 format from the library. For multi-chapter audiobooks, files are provided either individually or as a single archive (deferred to design).

## Acceptance Criteria

- Given an audiobook in the library, when the user clicks download, then they receive the audio file(s) in MP3 format
- Given a multi-chapter audiobook, when the user downloads it, then all chapter files are included
