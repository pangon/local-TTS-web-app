# REQ-F-library-listing: Audiobook Library Listing

**Type**: Functional

**Status**: Draft

**Priority**: Must-have

**Source**: [US-browse-audiobook-library](../user-stories/US-browse-audiobook-library.md)

**Source stakeholder**: [STK-end-user](../stakeholders.md)

## Description

The system shall display a list of all generated audiobooks, showing each audiobook's title, creation date, and chapter count.

## Acceptance Criteria

- Given the user opens the library view, when audiobooks exist, then all previously generated audiobooks are listed
- Given an audiobook in the library list, then its title, creation date, and chapter count are visible
- Given no audiobooks have been generated, when the user opens the library view, then an empty state message is shown
