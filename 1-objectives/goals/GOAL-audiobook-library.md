# GOAL-audiobook-library: Audiobook Library and Playback

**Description**: Browse, play, and manage generated audiobooks from the web interface. Users should have a library view of all their audiobooks and be able to resume listening where they left off.

**Status**: Draft

**Priority**: Must-have

**Source stakeholder**: [STK-end-user](../stakeholders.md)

## Success Criteria

- [ ] Users can browse a library of all previously generated audiobooks
- [ ] Users can play audiobooks directly from the web interface
- [ ] Users can resume playback from where they left off in a previous session
- [ ] Users can delete audiobooks they no longer need
