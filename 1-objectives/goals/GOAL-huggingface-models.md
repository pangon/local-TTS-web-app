# GOAL-huggingface-models: HuggingFace Open-Weight Model Support

**Description**: Support open-weight models from HuggingFace, loaded on-demand rather than pre-bundled. Users can choose from available models and the system handles downloading, caching, and loading them locally.

**Status**: Draft

**Priority**: Must-have

**Source stakeholder**: [STK-end-user](../stakeholders.md), [STK-developer](../stakeholders.md)

## Success Criteria

- [ ] Users can select from available HuggingFace open-weight TTS models
- [ ] Models are downloaded and loaded on-demand (not shipped with the app)
- [ ] The system manages model storage and caching locally
