# US-browse-audiobook-library: Browse Audiobook Library

**As a** end user, **I want** to browse a library of my generated audiobooks, **so that** I can find and access any audiobook I've created.

**Status**: Approved

**Priority**: Must-have

**Source stakeholder**: [STK-end-user](../stakeholders.md)

**Related goal**: [GOAL-audiobook-library](../goals/GOAL-audiobook-library.md)

## Acceptance Criteria

- Given I open the library view, then I see a list of all previously generated audiobooks
- Given an audiobook in the library, then I can see its title, creation date, and chapter count
