# US-delete-audiobook: Delete Audiobook from Library

**As a** end user, **I want** to delete audiobooks from the library, **so that** I can free up storage space.

**Status**: Approved

**Priority**: Must-have

**Source stakeholder**: [STK-end-user](../stakeholders.md)

**Related goal**: [GOAL-audiobook-library](../goals/GOAL-audiobook-library.md)

## Acceptance Criteria

- Given an audiobook in the library, when I choose to delete it, then it is removed along with its audio files
- Given I request deletion, then the system asks for confirmation before deleting

## Derived Requirements

- [REQ-F-delete-audiobook](../requirements/REQ-F-delete-audiobook.md) — Delete audiobook with confirmation
