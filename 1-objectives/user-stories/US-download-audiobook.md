# US-download-audiobook: Download Audiobook from Library

**As a** end user, **I want** to download an audiobook from the library, **so that** I can listen to it outside the application.

**Status**: Draft

**Priority**: Should-have

**Source stakeholder**: [STK-end-user](../stakeholders.md)

**Related goal**: [GOAL-audiobook-library](../goals/GOAL-audiobook-library.md), [GOAL-browser-ui](../goals/GOAL-browser-ui.md)

## Acceptance Criteria

- Given an audiobook in the library, when I click download, then I receive the audio file(s) in MP3 format
